# Evidence Map — Adaptive Coherence Under Constraint

## Purpose
This file is the project’s credibility scaffold. Its job is to separate:
- what is relatively well-supported,
- what is promising but still emerging,
- and what is currently best treated as synthesis-level hypothesis.

## Evidence Tier Legend

### Established
Supported by broad literature, replicated mechanisms, or convergent findings strong enough to present as relatively stable background.

### Emerging
Supported by plausible mechanism and some direct evidence, but still limited by small studies, narrow replication, or incomplete translational confirmation.

### Proposed synthesis
A coherent interpretation built by integrating multiple lines of evidence. Useful and often testable, but not yet directly established as a whole.

2. Leaning too hard on single-lab GAA findings without flagging replication limits
3. Treating the n=1 as proof instead of structured signal
4. Letting the most interesting mechanistic bridges outrun the available human evidence
5. Failing to separate core framework from broader speculative horizon

## Writing Rule for the Main Paper
Whenever a section makes a strong claim, check it against this file:
- Can this be stated as established?
- Should this be framed as emerging?
- Is this actually a proposed synthesis that needs softer wording?

If uncertain, default one tier lower.
